Image Search-Abstraction-Layer API

A freeCodeCamp Javascript Back-End Challange: Image-Search-Abstraction-Layer

User stories:

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Usage:

  • https://image-search-abs-layer-fcc.herokuapp.com/search/cats
  • https://image-search-abs-layer-fcc.herokuapp.com/search/cats?offset=2
  • https://image-search-abs-layer-fcc.herokuapp.com/latest
